העלאה ל-Vercel (מתוך GitHub):
-------------------------------
1. ודא שה-repo מכיל את 3 הקבצים:
   - index.html
   - vercel.json
   - package.json

2. ב-Vercel, תחת Project Settings > Build & Output Settings:
   - Build Command: השאר ריק (או: echo done)
   - Output Directory: . (נקודה בלבד)
   - Framework Preset: Other

3. לחץ Redeploy — זהו!
